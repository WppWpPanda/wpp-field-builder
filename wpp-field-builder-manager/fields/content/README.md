# Поле: Контент — `WPP_Content_Field`

Выводит произвольный HTML-контент внутри формы. Полезен для разделителей, заголовков секций, описаний и информационных блоков.

## ⚙️ Параметры

| Параметр       | Тип         | Описание |
|----------------|-------------|----------|
| `type`         | string      | `'content'` |
| `label`        | string      | Заголовок контента (опционально) |
| `content`      | string      | HTML-содержимое блока |
| `wrap`         | boolean     | Оборачивать в контейнер (по умолчанию `true`) |
| `classes`      | array       | Дополнительные CSS-классы |
| `width`        | string      | Ширина: full, 1/2, 1/3 и т.д. |

## ✅ Примеры

### 1. Простой текстовый блок

```php
new WPP_Content_Field([
    'type' => 'content',
    'content' => '<p>Заполните все обязательные поля, отмеченные звёздочкой (*).</p>'
]);
```

### 2. Разделитель секции с заголовком

```php
new WPP_Content_Field([
    'type' => 'content',
    'label' => 'Персональная информация',
    'content' => '<hr>',
    'width' => 'full'
]);
```

### 3. Информационный блок с иконкой

```php
new WPP_Content_Field([
    'type' => 'content',
    'content' => '
        <div class="alert alert-info">
            <i class="fas fa-info-circle"></i>
            <strong>Важно!</strong> Все данные будут проверены службой безопасности.
        </div>
    ',
    'wrap' => true
]);
```

### 4. Блок с ссылками и форматированием

```php
new WPP_Content_Field([
    'type' => 'content',
    'label' => 'Документы',
    'content' => '
        <h4>Необходимые документы:</h4>
        <ul>
            <li>Паспорт гражданина РФ</li>
            <li>СНИЛС</li>
            <li>Документ о доходах (2-НДФЛ или справка по форме банка)</li>
        </ul>
        <p><a href="/documents-help" target="_blank">Подробнее о документах →</a></p>
    ',
    'width' => 'full'
]);
```

### 5. Секция без обёртки (inline)

```php
new WPP_Content_Field([
    'type' => 'content',
    'content' => '<span class="text-muted small">Нажимая кнопку "Отправить", вы соглашаетесь с условиями обработки персональных данных.</span>',
    'wrap' => false
]);
```

### 6. Комбинирование с другими полями

```php
$fields = [
    // Заголовок секции
    'section_title_1' => [
        'type' => 'content',
        'label' => 'Контактная информация',
        'content' => '<hr>',
        'width' => 'full'
    ],
    
    // Поля секции
    'phone' => [
        'type' => 'text',
        'element_type' => 'tel',
        'label' => 'Телефон',
        'required' => true
    ],
    
    'email' => [
        'type' => 'text',
        'element_type' => 'email',
        'label' => 'Email',
        'required' => true
    ],
    
    // Информационный блок
    'contact_notice' => [
        'type' => 'content',
        'content' => '
            <div class="alert alert-warning">
                <i class="fas fa-exclamation-triangle"></i>
                Указывайте только актуальные контактные данные!
            </div>
        ',
        'width' => 'full'
    ],
    
    // Заголовок следующей секции
    'section_title_2' => [
        'type' => 'content',
        'label' => 'Дополнительная информация',
        'content' => '<hr>',
        'width' => 'full'
    ]
];
```

## 📝 Примечания

- **Безопасность**: контент обрабатывается через `wp_kses_post()`, разрешены только безопасные HTML-теги
- **Обёртка**: параметр `wrap` позволяет выводить контент без стандартной обёртки поля
- **Заголовок**: параметр `label` выводится как `<h3>` над контентом
- **Гибкость**: можно использовать любой допустимый HTML и WordPress-функции

## 🔧 Особенности

- Не создаёт input-элементов (только отображение)
- Не участвует в валидации формы
- Не сохраняется в данные формы
- Идеально для визуального разделения секций
- Поддержка WordPress-шорткодов в контенте

## 🎨 Стилизация

Используйте стандартные Bootstrap-классы или свои:

```css
.wpp-content-label {
    font-size: 1.25rem;
    font-weight: 600;
    margin-bottom: 1rem;
}

.wpp-content-body {
    color: #6c757d;
}

/* Информационные блоки */
.wpp-content-field .alert {
    padding: 1rem;
    border-radius: 0.25rem;
}
```

## 💡 Советы по использованию

1. **Разделители**: используйте `<hr>` для визуального разделения секций
2. **Предупреждения**: применяйте Bootstrap-алерты для важных сообщений
3. **Навигация**: добавляйте ссылки на помощь или документацию
4. **Инструкции**: размещайте подсказки перед группами полей
5. **Брендинг**: вставляйте логотипы или баннеры при необходимости
